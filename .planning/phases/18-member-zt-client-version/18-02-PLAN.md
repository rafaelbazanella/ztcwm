<test_convention>
**LOAD-BEARING for this plan — re-read before writing any test assertion that mentions `·` or `—`.**

Project precedent (`src/pages/network-detail.test.ts:579`): assertions on the middle dot and em-dash in test-body strings are written as **6-character ASCII escape sequences inside single-quoted strings**, NOT as the literal Unicode glyphs.

The two forms produce the same runtime string but differ at the source-byte level:

- `'·'` — six raw bytes: `\`, `u`, `0`, `0`, `B`, `7`. **CORRECT.** Greppable as `grep -c '\\\\u00B7'`.
- `'—'` — six raw bytes: `\`, `u`, `2`, `0`, `1`, `4`. **CORRECT.** Greppable as `grep -c '\\\\u2014'`.
- `'·'` — one raw byte (the literal U+00B7 codepoint). **WRONG** — gates won't find it.
- `'—'` — one raw byte (the literal U+2014 codepoint). **WRONG** — gates won't find it.

Use the escape form **only inside test-body string literals**. The production-code Lit `html\`…\`` template inside `network-detail.ts` continues to use the **literal glyphs** `·` and `—` directly — that is the existing convention at `network-detail.ts:529` and the executor must preserve it for visual scanability of the rendered output.

**Concretely:**
- Production code (`src/pages/network-detail.ts`): write `html\`<span>—</span>\`` and `html\`<span>· ${display}</span>\`` — these contain the actual codepoints (one byte each in the source).
- Test code (`src/pages/network-detail.test.ts`): write `expect(html).toContain('·')` and `expect(html).toContain('—')` — these contain the six-character escape sequence in the source.

The acceptance-criteria grep gates below are calibrated to count exactly the planned occurrences of each escape form in `network-detail.test.ts` — if you change the test count or duplicate assertions, recount and update the gates locally before declaring done.

```typescript
private async loadData(): Promise<void> {
    if (!this.networkId) return;
    this.loading = true;
    try {
        const [network, membersResult, peers] = await Promise.all([
            networkService.getNetwork(this.networkId),
            memberService.listMembersWithPeers(this.networkId),
            nodeService.getPeers().catch(() => []),
        ]);
        this.network = network;

        // Build peer info map: prefer active IPv4 path; fall back to first active path of any family (Plan 14-03, D-15)
        const peerInfoMap = new Map<string, { physicalAddress?: string; isPhysicalAddressIPv6?: boolean; online: boolean }>();
        for (const peer of peers) {
            const activePaths = (peer.paths ?? []).filter(p => p.active && p.address);
            const ipv4Path = activePaths.find(p => isIPv4(p.address.split('/')[0]));
            const chosenPath = ipv4Path ?? activePaths[0];
            const physicalAddress = chosenPath ? chosenPath.address.split('/')[0] : undefined;
            const isPhysicalAddressIPv6 = chosenPath !== undefined && !ipv4Path;
            const hasRecentActivity = peer.paths?.some(p => {
                const lr = p.lastReceive ?? 0;
                const ls = p.lastSend ?? 0;
                const latest = Math.max(lr, ls);
                return latest > 0 && (Date.now() - latest) < 300000;
            }) ?? false;
            peerInfoMap.set(peer.address, { physicalAddress, isPhysicalAddressIPv6, online: hasRecentActivity });
        }

        // Enrich version-carrying members with physicalAddress, IPv6-only flag, and online status from peers.
        // Version field arrives pre-merged from memberService.listMembersWithPeers (D-10 / Phase 18 / D-07 detection).
        this.members = membersResult.data.map(m => {
            const nodeId = m.nodeId || m.id;
            const peerInfo = peerInfoMap.get(nodeId);
            return {
                ...m,
                ...(peerInfo?.physicalAddress ? { physicalAddress: peerInfo.physicalAddress } : {}),
                ...(peerInfo?.isPhysicalAddressIPv6 ? { isPhysicalAddressIPv6: true } : {}),
                online: peerInfo?.online ?? false,
            };
        });

        this.editName = network.name || '';
        logService.info(`Loaded network ${this.networkId}`);
    } catch (err) {
        logService.error(`Failed to load network ${this.networkId}`, String(err));
    } finally {
        this.loading = false;
    }
}
```
